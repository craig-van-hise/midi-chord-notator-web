## 1. Project Context & Objectives
We are addressing three UX refinements.
1. **Dumb Pipe MIDI Provider:** Legacy logic in `MIDIProvider.tsx` is swallowing `NoteOff` events during Hold Mode, causing infinite audio sustain. We must remove all `heldModePendingNoteOffs` logic.
2. **Dynamic PLAY Envelopes:** The `PLAY` action must behave like a physical key, using hardware velocity and sustaining exactly as long as the hardware key (or pointer) is held down.
3. **MIDI Learn Persistence:** Configuration assignments must be saved to and loaded from `localStorage`.
