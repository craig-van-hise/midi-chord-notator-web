## 1. Project Context & Objectives
**Project:** Grand Staff MIDI Notator
**Current State:** The filter architecture is destructive and downstream. It overwrites the underlying musical data when notes go out of bounds, preventing them from returning when transformed back into range. It also fails to intercept live inputs, lacks a global hard limit, and breaks pitch-class identities using arbitrary modulo math for wrapping.
**Objective:** Execute a fundamental architectural rebuild of the routing pipeline. You will bifurcate the state into a `theoretical` layer (unbounded math) and a `safe` layer (filtered/clamped output), enforce strict modulo-12 pitch preservation, and universally route all MIDI input (live and calculated) through this new pipeline.

## 2. Technical Decisions & Dependencies
* **State Bifurcation (The Non-Destructive Lens):** * The transformation math must operate EXCLUSIVELY on a `theoreticalNotes` array (which can legally contain values like `-15` or `140`). This array is preserved in memory/refs.
  * The visual React state (SMuFL canvas/main keyboard) and the audio engine (Tone.js) must ONLY receive the `safeNotes` array, which is the output of `applyNoteFilter(theoreticalNotes)`.
* **Universal Gatekeeping:** Every MIDI Note On/Off from the physical controller, UI keyboard, and transformation engine must hit the `theoreticalNotes` array first, trigger the filter, and then propagate the `safeNotes` downstream.
* **Pitch-Class Preservation (Modulo 12 Strictness):** The `wrap` and `octave_wrap` modes must be combined or strictly redefined. Any out-of-bounds note MUST ONLY be shifted in exact multiples of 12 (`+/- 12`). If the user's selected range is narrower than 11 semitones (making an octave shift impossible), the filter must gracefully degrade to `block` for that note to prevent infinite loops.
* **Global Hard Limit:** Regardless of the user's filter settings, the final return statement of `applyNoteFilter` MUST strip any notes `< 0` or `> 127` to mathematically guarantee Tone.js and SMuFL never crash.
