## 1. Project Context & Objectives
The baseline collision detection and accidental spacing logic has been successfully restored via Git. The objective is now to safely implement cohemitonic cluster notation (3+ adjacent semitones/seconds) **without** modifying the internal logic of the restored baseline functions. 

We must implement a non-destructive pipeline:
1. Identify the topological `xLevel` of each note.
2. Apply the baseline logic to levels 0 and 1.
3. Dynamically calculate the extended X-offset for levels $\ge 2$, injecting exact spatial allowances for accidentals.
4. Pass the final X-coordinates into the restored accidental left-sweep algorithm.

## 2. Technical Decisions & Dependencies
**The Pipeline Architecture:**
You are strictly forbidden from altering the inner workings of the baseline Level 0/Level 1 offset logic or the baseline accidental left-sweep loop. You will construct a wrapper pipeline that operates in distinct passes.

**Dynamic Offset Mathematics (For Levels $\ge$ 2):**
Let `W_n` = Standard Notehead Width.
Let `W_a` = Standard Accidental Width.
Let `P` = Visual Padding (e.g., 0.2 * staff-space).

For any level $L \ge 2$, the starting X-coordinate must be calculated dynamically based on the accumulated width of the preceding levels to ensure no overlap. 
* **Formula:** $X_L = X_{L-1} + W_n + P + (\text{if level } L \text{ has an accidental } ? W_a : 0)$
* *Implementation Note:* This ensures that if the 3rd note requires a sharp, we physically push the notehead further right by the exact width of that sharp so the sharp does not bleed into the $L=1$ notehead.
