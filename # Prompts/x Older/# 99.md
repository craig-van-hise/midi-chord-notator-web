# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
This sprint synchronizes the global Chord Speller engine and Keyboard visuals with the user's explicit Write Mode spelling overrides. 

**The Bug:**
Currently, when a user locks a note to a specific spelling (e.g., writing an F# instead of a Gb), the `chord_speller.ts` engine ignores the visual choice because it only receives raw MIDI pitches. It subsequently labels the chord as "Gb" and the Keyboard renders "Gb, Bb, Db".

**The Solution:**
1. **String Translation:** `notationMath.ts` will convert the canvas's visual properties (`stepOffset` + `accidental`) into a theoretical string name (`"F#"`).
2. **Override Injection:** The `chordSpeller` pipeline will be updated to accept an `overrides` dictionary. If the calculated root of a chord has a user-defined override, the speller will adopt the override and base all interval math off the new root name.
3. **Dumb Client:** `Keyboard.tsx` will stop calculating its own spellings and instead render the exact strings broadcasted by the `NotationCanvas`.

## 2. Technical Decisions & Dependencies
* **Data Flow:** You will expand `ActiveNoteData` in `NotationCanvas.tsx` to include `spellingOverride?: string` (the explicit string from Write Mode) and `spellingString?: string` (the final calculated string to broadcast to the keyboard).
* **Pipeline Hooking:** You will modify `getChordSpelling` and `getChordSymbol` in `chord_speller.ts` to accept `overrides?: Record<number, string>`.

  2. Implement the following pure function:
     ```typescript
     export function getNoteNameFromPosition(stepOffset: number, accidental: string | null, keySignature: string): string {
         const letters = ['C', 'D', 'E', 'F', 'G', 'A', 'B'];
         const scaleStep = ((stepOffset % 7) + 7) % 7;
         const letter = letters[scaleStep];
         
         let accStr = '';
         if (accidental === SMuFL.accidentalSharp) accStr = '#';
         else if (accidental === SMuFL.accidentalFlat) accStr = 'b';
         else if (accidental === SMuFL.accidentalDoubleSharp) accStr = 'x';
         else if (accidental === SMuFL.accidentalDoubleFlat) accStr = 'bb';
         else if (accidental === SMuFL.accidentalNatural) accStr = '';
         else {
             // Fallback to diatonic accidental if null
             const diatonicMap = getDiatonicMap(keySignature);
             for (const [pc, data] of diatonicMap.entries()) {
                 if (data.step === scaleStep) {
                     if (data.acc === SMuFL.accidentalSharp) accStr = '#';
                     if (data.acc === SMuFL.accidentalFlat) accStr = 'b';
                     break;
                 }
             }
         }
         return letter + accStr;
     }
     ```
