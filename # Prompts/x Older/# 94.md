# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
The recent Write Mode and History Engine implementation introduced several logical bugs due to poor event-listener hierarchy and incomplete state-hooking. 

**The Bugs:**
1. Global shortcuts (`Esc`, `Shift + W`, `Cmd+Z`) do not fire unless a note is selected.
2. The `undoStack` is unpredictable and fails to capture discrete canvas writes, deletions, and toggle-mode insertions.
3. The `#ghost-note` element renders at a default (often extreme) Y-coordinate when Write Mode is first entered, requiring a mouse nudge to snap into place.

**The Solution:**
We must restructure `handleKeyDown` to evaluate global shortcuts *before* the selection gate. We must introduce a `lastPointerCoords` ref to track the mouse globally so the ghost note can instantly render at the correct position upon entering Write Mode. Finally, we must meticulously inject `commitState()` into every discrete mutation branch.

## 2. Technical Decisions & Dependencies
* **Event Hierarchy Mandate:** `handleKeyDown` must be split into two distinct sections: **Global Shortcuts** (Undo/Redo, Escape, Write Toggle) and **Selection Shortcuts** (Delete, Transpose).
* **Pointer Tracking:** You will use a new `useRef` to track the last known Y coordinate of the pointer across the canvas, enabling instant ghost-note snapping on double-click or `Shift+W`.
* **Deep Clone History:** Ensure `commitState` uses `activeNotes.current.map(n => ({ ...n }))` to prevent reference mutations from retroactively altering history states.

```typescript
const snapGhostNote = (clientY: number, rect: DOMRect) => {
    const pointerY = clientY - rect.top;
    const canvasCenterY = rect.height / 2;
    const relativeY = canvasCenterY - pointerY;
    let stepOffset = 0;

    if (relativeY >= 0) {
        stepOffset = Math.round((relativeY - staffSpace) / (staffSpace / 2));
    } else {
        stepOffset = Math.round((relativeY + staffSpace) / (staffSpace / 2));
    }

    const snappedY = canvasCenterY - (((stepOffset) * (staffSpace / 2)) + (relativeY >= 0 ? staffSpace : -staffSpace));

    const ghost = document.getElementById('ghost-note');
    if (ghost) {
        ghost.classList.remove('hidden');
        ghost.style.top = `${snappedY}px`;
        ghost.dataset.step = stepOffset.toString();
    }
};

```
