# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
This sprint enhances the "Write Mode" environment by making the ghost note adaptive to the active key signature and introducing an Accidental Override toolbar.

**Objective 1: Adaptive Diatonic Ghost Notes**
When Write Mode is active, hovering over a staff position must dynamically display the correct diatonic accidental based on the current key signature.

**Objective 2: Accidental Overrides**
A new toolbar must appear below the Key Signature selector, allowing the user to cycle through Flat (b, bb, off), Natural (natural, off), and Sharp (#, x, off) overrides. These overrides are mutually exclusive.

**Objective 3: Deterministic Key-Based Spelling (Spelling Lock)**
Notes inserted via Write Mode must be perfectly sensitive to the Key Signature and the Toolbar Override. Once placed, their spelling must be locked. The `chordSpeller` pipeline must be patched to respect this lock, ensuring that vertical chord context never overwrites the user's explicit written notation.

## 2. Technical Decisions & Dependencies
* **Data Model:** You will expand the `ActiveNoteData` interface in `NotationCanvas.tsx` to include an `isLockedSpelling?: boolean` flag.
* **Math Offloading:** I have pre-calculated the pure function required to convert a `stepOffset`, `keySignature`, and `overrideState` into a precise MIDI pitch and SMuFL accidental. You will implement this in `notationMath.ts`.
* **Pipeline Preservation:** `updateSpellings()` will still evaluate all notes to determine the chord symbol, but it will bypass modifying the `stepOffset` and `accidental` properties of any note flagged with `isLockedSpelling`.

### Phase 1: The Adaptive Pitch Engine (`notationMath.ts`) — ✅ Completed
* **Objective:** Implement the pure function that calculates the ghost note's target pitch and accidental based purely on the Key and Overrides.
* **Instructions:**
  1. Open `src/utils/notationMath.ts`.
  2. Export the new type at the top of the file:
     ```typescript

     ```
  3. Implement the following pure function at the bottom of the file:
     ```typescript
     export function calculateWriteModePitch(
         stepOffset: number, 
         keySignature: string, 
         override: AccidentalOverride
     ): { midiNote: number, accidental: string | null } {
         const scaleStep = ((stepOffset % 7) + 7) % 7;
         const octaveOffset = Math.floor(stepOffset / 7);
         let targetOctave = 4 + octaveOffset;
         const basePCs = [0, 2, 4, 5, 7, 9, 11]; // Diatonic steps for C, D, E, F, G, A, B

         // 1. Handle Explicit User Overrides
         if (override) {
             const naturalPitch = ((targetOctave + 1) * 12) + basePCs[scaleStep];
             let offset = 0;
             let accGlyph = null;
             if (override === 'b') { offset = -1; accGlyph = SMuFL.accidentalFlat; }
             if (override === 'bb') { offset = -2; accGlyph = SMuFL.accidentalDoubleFlat; }
             if (override === 'n') { offset = 0; accGlyph = SMuFL.accidentalNatural; }
             if (override === '#') { offset = 1; accGlyph = SMuFL.accidentalSharp; }
             if (override === 'x') { offset = 2; accGlyph = SMuFL.accidentalDoubleSharp; }
             return { midiNote: Math.max(0, Math.min(127, naturalPitch + offset)), accidental: accGlyph };
         }

         // 2. Diatonic Fallback (Adaptive to Key)
         const diatonicMap = getDiatonicMap(keySignature);
         let targetPC = 0;
         let diatonicAcc = null;
         for (const [pc, data] of diatonicMap.entries()) {
             if (data.step === scaleStep) {
                 targetPC = pc;
                 diatonicAcc = data.acc;
                 break;
             }
         }

         // Reverse Octave Boundary Corrections for calculation
         if (targetPC === 11 && scaleStep === 0) targetOctave -= 1;
         if (targetPC === 0 && scaleStep === 6) targetOctave += 1;

         const diatonicMidi = ((targetOctave + 1) * 12) + targetPC;
         return { midiNote: Math.max(0, Math.min(127, diatonicMidi)), accidental: diatonicAcc };
     }
     ```
