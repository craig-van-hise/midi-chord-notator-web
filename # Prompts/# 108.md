# Product Requirements Prompt (PRP) — ✅ Completed

## 1. Project Context & Objectives
**Objective:** Architect a robust Preview Voice Manager to eliminate orphaned notes during rapid navigation, and expand audio preview support to all multi-selection interactions.
**Current State:** Rapid keyboard traversal and transposition cause infinite sustain because previous notes are not explicitly released before new notes are triggered. Multi-selection actions (marquee, shift-click) do not trigger audio previews.
**End State:** A centralized `activePreviews` reference will track and aggressively release any currently playing preview notes before a new selection is auditioned. All selection methods (pointer clicks, marquee drags, and key commands) will pipe arrays of note strings into this manager to ensure full chords are previewed accurately.

## 2. Technical Decisions & Dependencies
* **Preview Voice Manager:** The worker agent MUST implement a `useRef<Map<string, ReturnType<typeof setTimeout>>>` to track which notes are currently previewing.
* **Interrupt Logic:** Before auditioning a new selection or transposition, the manager MUST iterate over the map, call `clearTimeout`, explicitly call `audioEngine.releaseNote` for the orphaned string, and clear the map.
* **Centralized Extractor:** After any selection mutation, the agent MUST map the `selectedNoteIds.current` set back to their physical string pitches via `activeNotes.current` and pass the array to the Voice Manager.

## 3. Implementation Phases

### Phase 1: The Preview Voice Manager — ✅ Completed
* **Objective:** Implement a state-safe helper function that handles forced note-offs for previous selections.
* **Instructions:**
  1. Open `src/components/NotationCanvas.tsx`.
  2. Near the top of the component, add: `const activePreviews = useRef<Map<string, ReturnType<typeof setTimeout>>>(new Map());`
  3. Replace the previous single-note preview helper with this exact robust implementation:
     ```typescript
     const playPreviewNotes = (noteStrings: string[], interrupt: boolean = true) => {
         if (interrupt) {
             // Force note-offs for anything currently previewing
             activePreviews.current.forEach((timeoutId, noteStr) => {
                 clearTimeout(timeoutId);
                 audioEngine.releaseNote(noteStr);
             });
             activePreviews.current.clear();
         }

         noteStrings.forEach(noteStr => {
             audioEngine.noteOn(noteStr, 0.67);
             const timeoutId = setTimeout(() => {
                 audioEngine.releaseNote(noteStr);
                 activePreviews.current.delete(noteStr);
             }, 500);
             activePreviews.current.set(noteStr, timeoutId);
         });
     };
     ```
