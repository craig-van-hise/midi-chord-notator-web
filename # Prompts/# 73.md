## 2. Technical Decisions & Dependencies
* **Font Resolution (Vite):** In Vite, assets placed in the `public/` directory are served at the root path (`/`). The CSS file must reference them via an absolute root path (`url('/fonts/Bravura.woff2')`), not relative paths or by traversing up to the `public` folder.
* **React Re-rendering:** The `forceUpdate` paradigm is an anti-pattern in modern functional React. If the canvas needs to re-render after a mutable Ref change (like selection), it should trigger the existing imperative rendering pipeline (e.g., `recalculateLayout()` or `clearAndRedraw()`), or use a proper dummy state trigger (`const [, setTick] = useState(0)`).
